Publisherss Input Form

 

Available options for managing Publisherss database table:

 

Add New Record

 

Click the Add New button to add a record to the Publisherss database.  Type in the Publishers Name (required), then Year Born (optional).  Click Save to save the new record; click Cancel to ignore the new record.

 

Edit Record

 

Click the Edit button to edit the displayed record.  Make any needed changes.  The Publishers Name is required and the Year Born is optional.  Click Save to save the changes; click Cancel to ignore the changes.

 

Delete Record

 

Click the Delete button to delete the displayed record.

 

Exit Program

 

Click the Done button to quit the application.
